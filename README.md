# VenuEase – Venue Booking Application

VenuEase is an Android application designed to simplify the process of searching, viewing, and booking venues for various events. The app provides a user-friendly interface where users can explore available venues and make bookings based on their requirements.

## Features
- Browse a list of available venues
- View venue details including price, location, and capacity
- User login and registration system
- Venue booking functionality
- Clean and simple user interface

## Technologies Used
- Android Studio
- Java
- XML
- Firebase / Local Database (if applicable)

## How to Run the Project
1. Clone or download the repository:
   git clone https://github.com/arunima-h/VenueEase.git

2. Open the project in Android Studio
3. Allow Gradle to sync the project
4. Connect an emulator or physical Android device
5. Click the Run button in Android Studio to launch the app
